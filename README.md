# Projeto Pachesco

> Status do projeto: desenvolvimento

### Instalação e inicialização: 
    git clone https://github.com/LUHK77/Pachesco.git
    cd Pachesco
    bash menu.sh


### Informações

O projeto Pachesco software de instalação de programas em linux utilizando shell script, tendo o intuito facilitar a instalação de arquivos e administração dos mesmos, projeto desenvolvido em grupo no IFRS Campus Osório.
