```
git clone https://github.com/LUHK77/Pachesco.git
cd Pachesco
bash menu.sh
```

## Descrição

**PACHESCO** é uma ferramenta interativa de linha de comando para gerenciamento e instalação de programas no ambiente Linux. Com uma interface simples exibida diretamente no terminal. 

A ferramenta oferece algumas sugestões de programas para desenvolvedores, designers e usuários comuns, além de permitir ao usuário especificar e escolher qualquer programa disponível dentro do repositório oficial do Linux. O instalador permite apenas os gerenciadores de pacotes APT e SNAP. No entanto, quando o usuário escolhe uma opção além das sugestões padrão a única opção de gerenciador de pacote disponível será o APT.

### Funcionalidades

O menu principal oferece quatro opções:

| Opção                             | Descrição                                                                                                  |
| --------------------------------- | ---------------------------------------------------------------------------------------------------------- |
| `1` - Instalar programas          | Instala novos softwares no sistema                                                                         |
| `2` - Biblioteca                  | Gerencia e consulta pacotes disponíveis                                                                    |
| `3` - Exportar Lista de Programas | Salva um registro dos programas instalados                                                                 |
| `4` - Importar Lista de Pacotes   | Recebe uma lista de pacotes com nome, tipo instalação e realiza a instalação de todos os pacotes listados. |

## Objetivo

O objetivo do projeto é simular, através de linhas de comando em Shell Script, uma biblioteca de gerenciamento de programas. Facilitando para que usuários inexperientes a respeito de comandos de terminal do Linux possam realizar instalação de programas.
